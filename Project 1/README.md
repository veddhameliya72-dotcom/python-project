# 🚀 Overview

This project is a simple yet educational **Python Command-Line Interface (CLI)** application designed to help beginners understand the fundamentals of Python programming.

The program interacts with the user, collects personal information through terminal input, and demonstrates:

- Python variable types
- Runtime memory allocation
- Type casting
- Input handling
- Basic arithmetic operations
- Output formatting

It is an excellent starter project for learning how Python manages different data types internally.

---

# ✨ Features

- 📥 Interactive terminal-based user input system
- 🔍 Automatic datatype identification using `type()`
- 🧠 Memory address visualization using `id()`
- 🔢 Integer, float, and string handling
- 🎯 Real-time birth year approximation
- 💻 Beginner-friendly Python syntax
- ⚡ Lightweight and dependency-free
- 🧪 Educational demonstration of Python internals

# 🧩 Project Architecture

```text
User Input
    ↓
Type Conversion
    ↓
Variable Storage
    ↓
Datatype Inspection
    ↓
Memory Address Extraction
    ↓
Formatted Output to Console
```

The application follows a simple sequential execution workflow, making it ideal for beginners learning procedural programming.

# ▶️ Usage

The application will prompt users to enter:

- Name
- Age
- Height
- Favourite Number
- Hobby
- Favourite Color

After submission, the program:

- Displays each variable
- Shows the datatype
- Prints the memory address
- Calculates approximate birth year
